# My dotfiles
These are my dotfiles. Feel free to copy, redistribute and even suggest me better configurations.  
It's my intention to follow KISS philosophy for both dotfiles and applications relying on them: I want all of them to be lightweight, few, well documented an bloat-free, so don't be suprised if you don't see me using oh-my-zsh 😉  

At the moment, I'm using [GNU Stow](https://www.gnu.org/software/stow/) to manage symlinks.
